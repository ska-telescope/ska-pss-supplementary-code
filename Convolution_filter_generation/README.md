# Convolution filters for fourier domain acceleration search

Python scripts to generate, plot, and test the convolution filters for PSS-FDAS.

Generating filterset for PSS-FDAS

=================================

The script `Generate_filters.py` can generate filters (43 filters in a single binary file) for FPGA-FDAS. The default values of the input parameters are set to the PSS-FDAS requirements.

usage: `python Generate_filters.py [-h] [-num_filters NUM_FILTERS] [-fil_sep FIL_SEP] [-fft_len FFT_LEN] [-outfile OUTFILE]`  


Plotting diagnostic plots to understand convolution filters

===========================================================

`Filter_analysis.py` can produce a set of plots that were used in the documentation of FDAS-convolution filter.

usage: `python Filter_analysis.py`


Comparing generated filters against CPP and MATLAB generated filters

====================================================================

The script `Compare_final_filters.py` can plot filters from the binary files written out by the python script with the filters generated by MATLAB and C++ script.

uses: `python Compare_final_filters.py [-h] -filterset1 FILTERSET1 -filterset2 FILTERSET2`
